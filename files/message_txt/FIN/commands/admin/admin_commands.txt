Lista komennoista:
\n/mieltymykset - määritä botin ja lounaan kieli
\n/tietoa - tietoa lounasbotista
\n/lotto - lounasloton tila
\n/liity - liity mukaan lounaslottoon
\n/peru - peru osallistumisesi lounaslottoon
\n\nAdmin commands:
\n/schedule_lottery [start day] [result day] - schedule monthly lottery with given start and result day
\n/pause_lottery - pause monthly scheduled lunch lotteries
\n/start_lottery - run lunch lottery result immediately
\n/get_log - get log file
